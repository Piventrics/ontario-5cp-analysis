Real-Time Data Sources & Collection Methods

🌐 Web Scraping

Direct extraction of current rates from official province websites using BeautifulSoup and automated HTTP requests.

⚡ Real-Time Updates

Alberta AESO updates hourly, Ontario IESO updates hourly, BC Hydro updates daily. All data is collected fresh from source.

🔒 Data Validation

Multiple data sources per province to ensure accuracy. Rates are extracted using regex patterns and validated for consistency.

📊 Live Monitoring

Dashboard automatically refreshes data and shows collection status. Historical tracking available for trend analysis.
